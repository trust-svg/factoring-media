# マイクロビジネス立ち上げ計画
**目標：月30〜50万円 | 強み：Web制作・広告運用・eBay輸出**

---

## 推奨ビジネスモデル（優先順位順）

### 1位：Web制作 × 広告運用 ワンストップ支援（最推奨）

**概要**
中小企業・個人店舗に対して、Webサイト制作と広告運用をセットで提供するサービス。「作って終わり」ではなく「作って集客まで責任を持つ」が強み。

**収益シミュレーション（月50万円）**

| 収益源 | 単価 | 件数/社数 | 月収 |
|---|---|---|---|
| LP・コーポレートサイト制作 | 20〜50万円/件 | 1〜2件 | 20〜50万円 |
| 広告運用顧問（月額） | 5〜10万円/社 | 5〜8社 | 25〜80万円 |
| サイト保守・更新（月額） | 1〜3万円/社 | 5〜10社 | 5〜30万円 |

> **月30万円の最小構成例：**
> LP制作1件（20万）+ 広告運用2社（5万×2）= 30万円
> **月50万円の目標構成：**
> 制作1件（30万）+ 広告運用4社（5万×4）= 50万円

**なぜこのモデルが最強か**
- Web制作だけだと案件が一時的な収入で終わる → 広告運用で**ストック収入化**
- 中小企業は「制作会社」と「広告代理店」を別々に使うのが面倒 → ワンストップが差別化
- 広告費の20%が相場の手数料。月50万円の広告費を持つクライアント5社 = 月50万円
- 初期投資ほぼゼロ、スキルがそのまま商品になる

**競合との差別化ポイント**
- 大手代理店：最低広告費が高い、担当が変わる、融通が利かない
- 個人フリーランス：制作か広告どちらか一方のみ
- **あなた：制作から運用まで一気通貫、中小企業に寄り添う伴走型**

### 2位：eBay輸出ビジネス（並行して展開）

**概要**
日本の中古品・コレクター向け商材を海外（主に米国・欧州）に販売。

**収益シミュレーション（月30万円）**

| 項目 | 数値 |
|---|---|
| 想定利益率 | 25〜50%（ニッチ商材なら60%超も） |
| 月商100万円 → 利益 | 25〜50万円 |
| 1商品平均単価（USD） | $30〜$150 |
| 必要な月間販売数 | 30〜100件 |

**売れるニッチカテゴリ（2025年版）**
1. トレーディングカード（遊戯王・ポケモン）：YoY 2.35倍成長
2. アニメ・ゲームフィギュア（絶版・限定品）
3. 中古カメラ・レンズ（Made in Japan ブランド力）
4. 腕時計パーツ・工具
5. 和楽器・民芸品

**メリット**
- 円安（2025年は1ドル145〜155円台）で有利な為替環境
- 既にeBay経験があるため即スタート可能
- Web制作×広告スキルで**自分の出品ページのクオリティを上げられる**（競合優位）

**課題**
- 仕入れ資金が必要（月商100万円なら仕入れ50〜70万円）
- 全世界のセラーと競合
- 在庫リスクあり

### 3位：越境EC出品代行サービス

**概要**
eBayへの出品経験を活かして、日本の中小メーカー・職人・古物商の代わりにeBayへ出品・販売代行を行う。

**収益モデル**
- 販売額の10〜20%を手数料として受け取る
- または月額固定5〜10万円 + 成果報酬

**強み**
- 自分でリスクを負わずに越境ECのノウハウを収益化
- クライアントが増えるほどスケールする

## 集客チャネル（優先順位順）

| チャネル | 特徴 | 初月の期待値 |
|---|---|---|
| **知人・紹介** | 最速・最高品質 | 1〜2件 |
| **クラウドワークス/ランサーズ** | 競合多いが即日提案可能 | 1〜3件 |
| **X（旧Twitter）** | 中長期でファン獲得 | 0〜1件 |
| **商工会議所** | 中小企業との直接接点 | 0〜1件/月 |
| **ポータルサイト掲載** | SEO集客の長期施策 | 2〜3ヶ月後 |

## リスクと対策

| リスク | 対策 |
|---|---|
| 案件が取れない | 単価を下げる・無料モニター募集・提案数を増やす |
| クライアントがいなくなる | 複数チャネルで集客・紹介の仕組みを作る |
| eBay在庫が売れない | ニッチカテゴリに特化・リサーチ精度を上げる |
| 収入が不安定 | 広告運用の月額ストック収入で安定基盤を作る |
