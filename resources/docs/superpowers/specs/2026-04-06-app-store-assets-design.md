# 仕様書 — 毎日すこやか診断所 アプリストア素材生成ワークフロー

**作成日**: 2026-04-06  
**アプリ名**: 毎日すこやか診断所  
**ターゲット**: 50代以上向け 性格診断＆交流マッチングアプリ

---

## ツール構成

```
ClaudeCode（オーケストレーター）
├── Flux Pro API        → 人物・背景素材生成
└── Figma MCP           → レイアウト組み立て・書き出し
```

**Flux Pro採用理由（Midjourney不採用）:**
- 正式API提供 → ToS違反リスクなし・自動化可能
- 従量課金 約$0.05/枚 → Midjourney月額固定より大幅安
- 品質はMidjourneyと同等レベル

## デザインスタイル — 「昭和モダンプレミアム」

昭和カラーのベースを保ちつつ、レイアウトとタイポはモダン高級感で統一。

| 要素 | 表現方法 |
|---|---|
| 背景 | グレイン（ノイズ）テクスチャ + ラジアルグラデーション |
| グラデーション | `#FFF4E6` → `#F28C63` 柔らかいラジアル |
| タイポグラフィ | 見出し: 明朝体 / 本文: ゴシック |
| 装飾 | 細い罫線・小さな花モチーフ（昭和喫茶風アクセント） |
| 人物写真 | ビネット加工 + 暖色フィルター（vintage warm film） |
| シャドウ | `#8B5E3C` 系ドロップシャドウ（柔らかめ）で奥行き |
| ボタン | `#8B5E3C` グラデーション + 内側ハイライトで立体感 |

## 実行フロー

```
Step 1: Flux Pro API で人物素材3点を生成
        - woman_a.png（40代女性）
        - woman_b.png（40代女性・別カット）
        - couple.png（50代男性+40代女性）

Step 2: Figma MCP でデザイントークン設定
        - カラー・フォント・エフェクト定義

Step 3: Figma MCP で全7フレームにレイアウト組み立て
        - 素材インポート → テキスト配置 → 装飾追加

Step 4: Figma MCP で仕様通りに全素材書き出し
        - 各フレームを正しいサイズ・形式で書き出し
```

## 元案からの変更点

| 元案 | 最終案 | 理由 |
|---|---|---|
| Midjourney | Flux Pro API | ToS違反リスク・コスト削減・自動化 |
| 1920×1080 | 1080×1920 | スマホ縦向き標準サイズ |
| 単色背景 | グレイン+グラデーション | リッチ感・昭和モダンプレミアム表現 |
| 通知/スプラッシュもFlux | Figmaのみ | 精度が高く工数ゼロ |
| 直接的な体型表現 | `naturally feminine, soft curves` | コンテンツフィルター回避・同等効果 |
