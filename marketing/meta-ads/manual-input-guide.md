# 手動広告作成ガイド — コピペ用

広告マネージャーで既存の広告セットに広告を追加する手順：
1. 広告マネージャー → 該当キャンペーン → 広告セットを選択
2. 「+作成」→ 広告を追加
3. 以下のテキストをコピペ → 画像を設定 → PAUSED で保存
