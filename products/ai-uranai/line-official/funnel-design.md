# LINE公式ファネル設計

## 全体フロー

```
LINE友達追加
    ↓
ウェルカムメッセージ（自動）
    ↓
無料占い体験（タロット1枚引き）
    ↓
── 3日以内に再利用 ──→ 継続ナーチャリング
    ↓
有料プラン案内（3日後〜）
    ↓
スタンダード or プレミアム転換
    ↓
毎朝配信 / 無制限利用（定着）
```

---

## メッセージフロー詳細

### 1. ウェルカムメッセージ（友達追加直後）

```
🌟 ようこそ！AI占いへ

毎日の気づきをAIがお届けします。

【今すぐ無料で試せる占い】
🃏 タロット占い（1日1回無料）
⭐ 今日の星座運勢
💫 相性診断

下のメニューから選んでスタート！
```

### 2. 無料体験後（即時）

```
鑑定結果はいかがでしたか？

毎日のメッセージを受け取りたい方は
スタンダードプランへどうぞ。

✅ 無制限占い
✅ 全カテゴリ解放
✅ ¥980/月（いつでも解約可）

[プランを見る]
```

### 3. ナーチャリング（3日後・未転換者）

```
こんにちは✨

今日もあなたのことが気になっていました。

今の状況に何かヒントが必要ですか？
タロットが答えます。

[今日の無料占いを見る]
```

### 4. アップセル（7日後・スタンダード会員）

```
【プレミアムプランのご案内】

毎朝7時にあなた専用の運勢をお届けします。

📱 毎朝プッシュ通知
🔮 詳細鑑定（通常の3倍のボリューム）
💬 何度でも深掘り質問OK

¥2,980/月（スタンダードより¥2,000UP）

[プレミアムに変更する]
```

## リッチメニュー設計

| ボタン | アクション |
|---|---|
| 🃏 タロット占い | タロットフロー起動 |
| ⭐ 今日の運勢 | 星座選択→運勢表示 |
| 💑 相性診断 | 相性フロー起動 |
| 🔢 数秘術 | 生年月日入力フロー |
| 💎 プランを見る | 料金ページ表示 |
| ❓ 使い方 | FAQ表示 |

---

## 転換率目標

| ステップ | 目標転換率 |
|---|---|
| 友達追加 → 無料体験 | 80% |
| 無料体験 → 有料転換 | 15% |
| スタンダード → プレミアム | 25% |
| 月次継続率 | 85% |
