## 技術スタック

| 用途 | 技術 |
|---|---|
| AI生成 | Anthropic Claude API (`claude-opus-4-5-20251101` or `claude-sonnet-4-5-20250929`) |
| LINE Bot | LINE Messaging API (`line-bot-sdk-python` v3系) |
| Webサーバー | FastAPI + Uvicorn |
| DB | SQLite（開発） → PlanetScale or Supabase（本番）|
| 決済 | LINE Pay または Stripe |
| ホスティング | Railway or Render（無料枠から始める）|
| 環境変数 | `.env` ファイル（絶対にコミットしない）|

---

## Claudeへのコーディング規則

1. **Python は PEP8 + 型ヒント必須**
2. **APIキー・シークレットは必ず `.env` から読む**（ハードコード禁止）
3. **LINE Webhook は署名検証を必ず実装する**
4. **占い結果は毎回 Claude API を呼ぶ**（キャッシュ禁止。体験の唯一性が価値）
5. **ユーザーIDはLINEのuserIdをそのまま使う**（個人情報は最小限）
6. **無料/有料プランの分岐は `user_plan` テーブルで管理する**

## 占いカテゴリ（実装する種類）

1. **タロット占い** — 1枚引き、3枚引き
2. **星座占い** — 今日の運勢、相性

4. **姓名判断** — 名前ベース

## 注意事項

- 占い結果は「エンターテインメント」として提供する旨を必ず明記
- 医療・法律・投資アドバイスに誘導するプロンプトは書かない
- 未成年への課金誘導はしない
- LINE利用規約に反するコンテンツは作らない
