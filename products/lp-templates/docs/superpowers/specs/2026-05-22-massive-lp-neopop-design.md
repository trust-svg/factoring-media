## 2. LP セクション構成（全10セクション）

### ① Hero
- **キャッチ**: 「まだいける。絶頂期は、今。」
- **サブ**: 累計マッチング数のチラ見せ（例：「累計18万人が登録」）
- **CTA**: LINEグリーンボタン「LINEで無料登録」
- **アニメ**: 3つの浮遊ブロブ（赤×黄グラデーション）がふわふわ動く（AOS + CSS animation）

### ② 実績バナー
- カウントアップアニメ（Intersection Observer でビューポートに入った瞬間に発火）
- 数字3点セット: 「累計18万人」「毎日320件マッチング」「満足度92%」
- 横一列のポップなカードデザイン

### ③ 共感セクション「こんな50代、いませんか？」
- 悩み例を箇条書きカード（「出会いの場がない」「マッチングアプリの使い方がわからない」「年齢が気になる」など）
- カードがスクロールで順番にフェードイン

### ④ なぜ50代はモテる・ヤレるのか
- トーン: 共感系 × データ寄り
- データカード例:
  - 「女性の68%が年上の男性を好む」
  - 「包容力・安定感・経済力が選ばれる理由No.1〜3」
  - 「30〜40代女性の4人に1人が年上限定」
- ポップなアイコン＋数字で展開、データごとにアニメで入場

### ⑤ シミュレーター「あなたのマッチング予測」（オリジナル要素）
- **フロー**: 年齢入力（スライダー）→ 職業タップ（4択）→ 趣味タップ（複数選択）→ 結果表示
- **結果**: 「1週間で推定 **○人** とマッチング可能！」（計算式: 基本値15 + 年齢ボーナス[50代:+3, 60代:+5] + 職業ボーナス[会社員:+2, 経営者:+8, 自営:+5, その他:+1] + 趣味ボーナス[選択数×2]。最小12〜最大47の範囲）
- 結果表示後に「今すぐLINEで試してみる →」CTAが出現
- アニメ: 結果数字がカウントアップしながら表示

### ⑥ 特徴 × 3点
- 完全匿名・プロフ非公開
- 50代以上限定の静かな空間
- 30〜45歳の女性と繋がれる
- ポップなアイコンカード、AOS フェードイン

### ⑦ 3ステップ「始め方はカンタン」
- LINE登録（3分）→ プロフィール設定 → マッチング開始
- ステップが左から順番にスライドイン

### ⑧ 体験談
- 成功事例カード3〜4件（「○○さん、54歳」形式）
- Swiper.js でスライド
- 各カードに女性の写真（生成画像）を使用

### ⑨ FAQ
- よくある質問 5〜7件
- アコーディオン形式（ CSS animate height）

### ⑩ 最終CTA
- 大きなLINEボタン
- 「すでに18万人以上が登録」テキストで背中押し
- 浮遊ブロブ再登場でHeroと統一感

## 4. 追加案（ユーザー提案への応答）

設計中に浮かんだ「合いそうな追加アイデア」:

1. **フローティングLINEボタン（固定）**: スクロール中も常に画面下部にLINEボタンを表示。スマホユーザーの離脱を防ぐ。
2. **「今アクティブな女性 ○人」バッジ**: ヒーロー付近にリアルタイム風のオンライン人数表示。緊急性・希少性を演出。
3. **ハートが飛ぶアニメ**: シミュレーターの結果表示時やCTA押下時にハートエフェクトが飛ぶ。ネオポップの遊び心を強調。

| スタイリング | CSS カスタムプロパティ + バニラCSS（Tailwind不使用、依存なし） |

| スライダー | Swiper.js（体験談セクション） |
| カウントアップ | Vanilla JS + Intersection Observer |
| フォント | Google Fonts（Noto Sans JP + Inter） |
| アイコン | Font Awesome 6 |
| 画像 | 生成画像（ChatGPT Image Pro or Nano Banana Pro） |
| ファイル | 単一 `index.html`（CSS/JS inline） |
| レスポンシブ | モバイルファースト、ブレークポイント 640px |
