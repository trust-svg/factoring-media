## 1. 全体方針

- 参照モックアップ: `products/invest-simulator/ui-mockups.html`

## 2. ヘッダー

- ロゴ + 市場開閉バッジ（🇯🇵 JP / 🇺🇸 US）をピル表示
- 更新ボタン（現行の↻ボタンを維持）
- 背景: 白、下ボーダー: `#e2e8f0`

---

## 3. KPI カード（3カラム）

現行の4カラム → 3カラムに変更。

| カード | アクセントカラー | 内容 |
|--------|-----------------|------|
| 総資産 | `#3b82f6`（青） | $値 + ¥換算 + 初期比% ピルバッジ |
| 含み損益 | `#10b981`（緑）または `#ef4444`（赤） | +$値 + ¥換算 + 全体% ピルバッジ |
| 現金残高 | `#f59e0b`（琥珀） | $値 + JP/US内訳 + 総資産比% ピルバッジ |

各カードは `border-top: 4px solid <アクセントカラー>` で色付け。  
損益カードは正負で緑/赤を切り替え。

---

## 4. 2カラムセクション（チャート + 銘柄損益バー）

```
[ 資産推移チャート (1.3fr) ]  [ 保有銘柄 損益率バー (1fr) ]
```

### 4-1. 資産推移チャート
- 現行の Chart.js 折れ線グラフを維持
- コンテナを `background:#f8fafc; border-radius:8px` で包む
- 直近データは濃い青（`#3b82f6`）、過去データは薄い青（`#bfdbfe`）

### 4-2. 保有銘柄 損益率バー
- 全保有銘柄をバー比較で表示
  - 損益率の絶対値に比例したバー幅（最大 = 最大損益率の銘柄が100%）
  - 含み益: `#3b82f6`（青）、含み損: `#fca5a5`（赤）
- ティッカー / バー / 損益% を横並び
- `/api/positions` の既存データをそのまま利用

## 5. 取引履歴テーブル

## 6. AI 分析ログ

- 現行の折りたたみ式カードを廃止 → 常時2行表示のコンパクトリストに変更
- 各エントリ: `background:#f8fafc; border-radius:8px; border-left:3px solid`
  - BUY: 左ボーダー `#3b82f6`
  - SELL: 左ボーダー `#f97316`
  - HOLD: 左ボーダー `#cbd5e1`
- ヘッダー行: バッジ + 時刻
- 本文: 2行クリップ（`-webkit-line-clamp:2`）
