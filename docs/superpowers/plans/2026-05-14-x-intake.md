### Task 2: X API call ヘルパー `~/.claude/scripts/x-intake-fetch.sh` 作成

**Files:**
- Create: `/Users/Mac_air/.claude/scripts/x-intake-fetch.sh`

- [ ] **Step 1: スクリプトを Write**

`/Users/Mac_air/.claude/scripts/x-intake-fetch.sh` を以下で作成:

```bash
#!/bin/bash
# X API v2 取得ヘルパー (spec #4)
# Usage: x-intake-fetch.sh <tweet-url-or-id>
# Output: JSON { id, text, author_username, created_at, rate_limit_remaining }
#         or { error: "rate_limit_exceeded" | "tweet_not_found" | ... }

set -euo pipefail

INPUT="${1:-}"
[ -z "$INPUT" ] && echo '{"error":"no_input"}' && exit 1

# URL なら tweet ID 抽出、数字だけならそのまま
TWEET_ID="$(echo "$INPUT" | grep -oE '[0-9]{15,25}' | head -1)"
[ -z "$TWEET_ID" ] && echo '{"error":"invalid_input"}' && exit 1

ENV_FILE="/Users/Mac_air/.claude/.env.x-api"
[ -f "$ENV_FILE" ] || { echo '{"error":"env_missing"}'; exit 1; }
# shellcheck disable=SC1090
source "$ENV_FILE"
[ -z "${X_API_BEARER_TOKEN:-}" ] && echo '{"error":"token_missing"}' && exit 1

# API call
TMP_HEADERS="$(mktemp)"
TMP_BODY="$(mktemp)"
HTTP_CODE="$(curl -s -o "$TMP_BODY" -D "$TMP_HEADERS" -w "%{http_code}" \
  -H "Authorization: Bearer ${X_API_BEARER_TOKEN}" \
  "https://api.x.com/2/tweets/${TWEET_ID}?tweet.fields=author_id,created_at&expansions=author_id&user.fields=username")"

RATE_REMAINING="$(grep -i '^x-rate-limit-remaining' "$TMP_HEADERS" | awk '{print $2}' | tr -d '\r' || echo unknown)"

case "$HTTP_CODE" in
  200)
    jq --arg rate "$RATE_REMAINING" '{
      id: .data.id,
      text: .data.text,
      created_at: .data.created_at,
      author_username: (.includes.users[0].username // null),
      rate_limit_remaining: $rate
    }' "$TMP_BODY"
    ;;
  429)
    echo "{\"error\":\"rate_limit_exceeded\",\"rate_limit_remaining\":\"${RATE_REMAINING}\"}"
    ;;
  404)
    echo '{"error":"tweet_not_found"}'
    ;;
  401)
    echo '{"error":"unauthorized"}'
    ;;
  *)
    echo "{\"error\":\"http_${HTTP_CODE}\",\"body\":$(jq -Rs . < "$TMP_BODY")}"
    ;;
esac

rm -f "$TMP_HEADERS" "$TMP_BODY"
```
